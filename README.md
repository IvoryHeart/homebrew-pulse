# homebrew-pulse

Homebrew tap for [pulse](https://github.com/IvoryHeart/pulse) — a macOS health monitor (CLI + menu bar app).

## Install

```sh
brew tap IvoryHeart/pulse
brew install pulse
```

Or in one command:

```sh
brew install IvoryHeart/pulse/pulse
```

## Bleeding edge (build from `main`)

```sh
brew install --HEAD IvoryHeart/pulse/pulse
```

## Updating

The formula in this tap is updated automatically by the release workflow in the [pulse](https://github.com/IvoryHeart/pulse) repository whenever a new `v*` tag is pushed. Manual edits are not expected.
